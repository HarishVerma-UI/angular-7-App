# Clone repo

Clone this repo into new project folder (e.g., `my-proj`).
```bash
$ git clone  https://github.com/harishwordpress/angular-7-App  my-proj
$ cd my-proj
```
# Install Dependency
```bash
make sure u have node or npm installed 
npm -v //5.6.0
node -v // v8.9.4

npm install
```

# Start the project 
```bash
npm Start 
npm run-script build
```
